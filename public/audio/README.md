# Tilawat audio

Place your recitation MP3 here as **`surah-yaseen.mp3`**.

Recommended:

- Short clip (1–3 minutes) or selected ayat — full Surah Yaseen is long for a background toggle
- File you own rights to use (your recording, or licensed audio)
- MP3 format, 128kbps is enough for web

Until you add this file, the app plays **Surah Yaseen ayat 1–12** online (~3–4 minutes), then stops. Replace with your own MP3 anytime.

Tilawat **auto-starts** after the loading screen. Some phones block autoplay until the first tap — then it starts on the first touch. The bottom-right button mutes/unmutes.
